Basic
---------------
{: .-three-column}

### Project

| Code          | Description                    |
| ------------- | ------------------------------ |
| `<SPC> p p`   | switch project                 |
| `<SPC> p a`   | add project                    |
| ---           | ---                            |
| `<SPC> <SPC>` | find file in project           |
| `<SPC> /`     | text search keyword in project |
| ---           | ---                            |

### Go Mode


| Code  | Description      |
| ----- | ---------------- |
| `g d` | go to definition |
| `g D` | go to reference  |
| ----- | ---------------  |
